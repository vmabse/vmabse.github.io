# Vagn & Maskin i Norden AB

Webbplats för Vagn & Maskin i Norden AB - transport och logistik i Östergötland och hela Sverige.

## Om sidan

Statisk webbplats hostad via GitHub Pages på [vm-ab.se](https://vm-ab.se).

## Teknisk stack

- HTML5
- [Tailwind CSS](https://tailwindcss.com/) (via CDN)
- Vanilla JavaScript

## Filstruktur

```
vmabse.github.io/
├── images/
│   ├── hero.jpg          # Hero-bild (lastbil)
│   ├── logo.png          # Huvudlogotyp (horisontell)
│   ├── logo-icon.png     # Logotyp-ikon (VM)
│   └── logo-alt.png      # Alternativ logotyp
├── index.html            # Webbplatsen
├── CNAME                 # Custom domain config
└── README.md
```
